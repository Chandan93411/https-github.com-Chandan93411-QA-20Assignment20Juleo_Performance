1.* SetUp Java 17 on your Machine
2.* Install Jmeter latest version on your machine
3.* Run the Jmeter Executable Jar or Jmeter window batch file bin directory of Jmeter folder
4.* Now Open the AmazonCheckout_500_user.jmx under Jmeter TestPlan
5.* Setup is done with 500 Users with 50 Ram-up period
6. * Under summary report you can check the throuput and error under all samples
7. * Under graph result you can check the Average , median , deviation and throughtput
8. * Under view result tree you can measure the network delay of each individual request.
